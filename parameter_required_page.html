List of runtime parameters required by modules.

This page gives information about module dependencies.

Module arn_arp
Parameters required by arn_arp submodule:
  1. CHKPOINTD:
    • IFCHKPTRST : if restart
    • CHKPTSTEP - checkpiont dump frequency (number of time steps)
  2. TIME_STEPPERD:
    • tst_mode :
      • 0 : no arnoldi
      • 1 : direct mode
      • 2 : adjoint mode
      • 3 : initial optimal condition
    • tst_step : frequency of calling arn_vsolve (number of time steps)
    • tst_cmax : maximal number of arnoldi cycles
    • tst_tol : PARPACK tolerance for Ritz values
  3. CONHT (for conjugated heat transfer):
    • CHCST_SC, CHCFF_V, CHCFF_T : velocity and temperature scaling factors
    • CHGRX, CHGRY, CHGRZ : gravitational acceleration
    • CHRA, CHPR : Rayleight and Prandtl numbers
  4. Nek5000 PARAM array
    • PARAM(66) - write format
    • PARAM(67) - read format
Module chkpoint_mstep
