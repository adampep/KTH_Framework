Detailed Description

Add white noise to velocity field at given time in a box.

This set of routines adds a white noise in the rectangular box performing global face averaging at the end. To keep the field independent on number of number of processors we use math_ran_dst insted or ran1 as a random number ganarator. At the end velocity field is smoothed at the element faces.
