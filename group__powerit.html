Detailed Description

Power iteration submodule.

Warning
This module is not upgraded to stable Nek5000 relese yet
Module interface:\n
Global interface list:
Interface provided to tstepper:
  • stepper_param_in
  • stepper_param_out
  • stepper_vinit
  • stepper_solve
Module parameters:\n
Global parameter list:
Parameters provided by power iteration submodule (include file - POWERITD):
  • PWIL2NORM : (real) initial L2 norm of the evolved vector for every cycle
Global parameter dependency:
Parameters required by power iteration submodule:
  1. CHKPOINTD:
    • IFCHKPTRST : if restart
    • CHKPTSTEP - checkpiont dump frequency (number of time steps)
  2. CONHT (for conjugated heat transfer):
    • CHCST_SC, CHCFF_V, CHCFF_T : velocity and temperature scaling factors
    • CHGRX, CHGRY, CHGRZ : gravitational acceleration
    • CHRA, CHPR : Rayleight and Prandtl numbers
Module parameter usage:\n
&powerit
pwil2norm= 1.0000000000000000 ,
